1. 寒假生活¶

1.1. 1.12¶

首先记录一下鼓捣了一晚上的Sphinx！ tmd现在我终于知道怎么用githubpages进行发布了！不过有以下几个需要注意的点：

1.1.1. Sphinx 发布学习¶

1.1.1.1. 方法一¶

直接在仓库进行发布，但是这样的话，一定要注意：

  • 把 build 文件夹名改为 docs

  • 在根目录下创建 .nojekyll 文件

  • 在 build 文件夹下，直接拷贝出原来 html 文件夹的所有内容，因为发布时需要目标文件夹下直接有 index.html 文件。

1.1.1.2. 方法二¶

使用自己的github站点进行发布，但是这个是单独属于自己的东西哎！注意点：

  • githubpages只能直接生成 html 文件，所以一定要把生成的所有内容复制到自己的专属文件夹下才能成功生成网页！
